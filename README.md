# GitHub PR Review Agent

AI-powered pull request review system that analyzes GitHub PR diffs using multiple review agents.

## Features

- Fetch GitHub Pull Requests
- Extract code diffs
- AI-powered code review
- Bug detection
- Style checking
- Test coverage suggestions
- Documentation review

## Tech Stack

- Node.js
- Express.js
- GitHub API
- Gemini AI
- React (planned)
